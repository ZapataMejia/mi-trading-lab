FON DEO EMA — INSTALACION EN STRATEGYQUANT X
============================================

IMPORTANTE: StrategyQuant NO permite subir una estrategia .sqx hecha fuera del programa.
El archivo .sqx solo lo genera SQX cuando TU guardas la estrategia desde AlgoWizard.
Lo maximo que se puede automatizar desde fuera son los SNIPPETS (indicador + condiciones).

PASO 1 — INSTALAR SNIPPETS (una vez)
------------------------------------
Opcion A (copiar carpetas — RECOMENDADO):
  1. En el Mac: carpeta trading/sqx/bundle/user/
  2. Por RDP al VPS: copia todo "user" dentro de:
     C:\SQX_143\user\
     Debe quedar:
       ...\user\extend\Snippets\SQ\Blocks\Indicators\Custom\LiquiditySweep\LiquiditySweep.java
       ...\user\extend\Snippets\SQ\Blocks\Indicators\Custom\FondeoEMAcross\...
  3. Code Editor -> Compile all -> reinicia SQX

  NOTA: El .java debe estar en Indicators\Custom\LiquiditySweep\
        (no en Indicators\LiquiditySweep\ suelto — el package es Custom)

Opcion A2 (solo LiquiditySweep):
  Copia la carpeta LiquiditySweep entera al VPS en la ruta de arriba.

Opcion B (Code Editor):
  1. Descomprime este zip.
  2. Code Editor -> Import/Export -> Import extensions (si tienes .sxp).
     Si no tienes .sxp, usa Opcion A.

  3. Code Editor -> Compile all (Compilar todo).
  4. Reinicia StrategyQuant si no aparecen los bloques.
